# Concept Diagram Switch Example

## 主题

达尔文 skill 不是一个工具，而是一套会自己进化的 skill 系统

## 视频类型

`concept-diagram-switch`

## 目标时长

`6s`

## 顶部结论层

- 文案：女娲负责造 skill，达尔文负责让 skill 一直进化
- 动效：`type-lite`

## 中部证据层

- `proof-01`
  - 系统总览图
  - 作用：说明“造 skill”和“进化 skill”是两套角色
- `proof-02`
  - 演化流程图
  - 作用：说明它不是一次生成，而是循环优化
- `proof-03`
  - 批量进化结果图或结构卡
  - 作用：说明系统输出会越来越强
- 切换方式：`crossfade + zoom-in settle`

## 底部解释层

- 文案：它的重点不是再做一个 skill，而是让 skill 进入可迭代状态
- 状态：固定

## 节奏

- `0.0 - 0.8s`
  - 标题入场
- `0.8 - 2.2s`
  - 证据 1
- `2.2 - 3.8s`
  - 证据 2
- `3.8 - 5.4s`
  - 证据 3
- `5.4 - 6.0s`
  - 概念停留
