# List Build Example

## 主题

AI Skill 的竞争，拼的不是技术，是分发效率

## 视频类型

`list-build`

## 目标时长

`12s`

## 顶部结论层

- 文案：AI Skill 最后的竞争，不是谁做得多，而是谁分发得更快
- 动效：`wipe-in + flow-in`

## 中部证据层

- 内容形式：逐条长出 4 个判断项
- 条目示例：
  - 不会写代码的人能不能 1 分钟用上
  - 模板有没有现成可调用入口
  - 使用门槛是不是低到能转发
  - 结果是不是能快速被看见
- 推进方式：`flow build`

## 底部解释层

- 文案：真正的优势不是把 skill 做出来，而是让普通人立刻用起来
- 状态：固定

## 节奏

- `0.0 - 1.0s`
  - 标题进入
- `1.0 - 3.0s`
  - 条目 1
- `3.0 - 5.0s`
  - 条目 2
- `5.0 - 7.0s`
  - 条目 3
- `7.0 - 9.0s`
  - 条目 4
- `9.0 - 12.0s`
  - 停留和补完
