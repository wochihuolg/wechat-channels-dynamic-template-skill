# Hook Proof Switch Example

## 主题

SkillSpector 为什么值得关注

## 视频类型

`hook-proof-switch`

## 目标时长

`6s`

## 顶部结论层

- 文案：暴涨 1.1 万 Star，不是因为它会写 Agent，而是因为它会查 Agent 技能漏洞
- 动效：`flow-in`

## 中部证据层

- `proof-01`
  - GitHub 仓库首页截图
  - 作用：证明项目热度和真实存在
- `proof-02`
  - 功能结构或扫描流程图
  - 作用：证明它不是概念，而是有明确能力
- `proof-03`
  - 漏洞结果页或检查结果卡
  - 作用：证明它能产出实际发现
- 切换方式：`blur-replace`

## 底部解释层

- 文案：它不是在教你造 skill，而是在帮你先看 skill 会不会出事
- 状态：全程固定
